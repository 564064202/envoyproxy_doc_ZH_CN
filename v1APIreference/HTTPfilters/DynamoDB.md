### DynamoDB
DynamoDB[配置概述](../../Configurationreference/HTTPfilters/DynamoDB.md)。

```
{
  "name": "http_dynamo_filter",
  "config": {}
}
```
- **name**<br />
	(required, string) 过滤器名称。目前唯一支持的值是`http_dynamo_filter`。

- **config**<br />
	(required, object) 该过滤器不使用任何配置。
